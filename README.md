# Ejercicios de DOM y BOM en JavaScript

## Descripción📝

Este proyecto consiste en la práctica de conceptos fundamentales de JavaScript, enfocados principalmente en:
Manipulación del DOM (Document Object Model)
Uso del BOM (Browser Object Model)
Programación orientada a objetos con clases
Se desarrolló una aplicación sencilla que permite crear una persona mediante un formulario y ejecutar distintas funciones sobre ella.

 ## Funcionalidades 🚀
- Crear una persona ingresando datos en un formulario
- Mostrar a qué generación pertenece
- Indicar si es mayor de edad
- Mostrar todos los datos ingresados
